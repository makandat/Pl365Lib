概要

Pl365Lib は Perl のプログラムを素早く作るためのライブラリである。現在のバージョンでは、コンソールアプリケーションと CGI を作ることができる。


モジュールリスト

  • Pl365Lib - Common　コンソールアプリケーションを関数を含むモジュール。
  • Pl365Lib - FileSystem　ファイル操作や入出力を行う関数を含むモジュール
  • Pl365Lib - Text　テキスト処理のための関数を含むモジュールを含む。
  • Pl365Lib - DateTime　日付時刻処理のためのクラス。
  • Pl365Lib - MySQL　MySQL を簡単に使うためのクラス。
  • Pl365Lib - WebPage　CGI を簡単に作れるクラス。
  • Pl365Lib - Parameters　コマンドラインパラメータを簡単に扱うためのクラス
  • Pl365Lib - Regexp　ほかの言語がサポートするような関数で正規表現を扱うためのクラス

ユーティリティ

  • utperl (ランチャ)
  • shtml.pl (HTML 作成ツール)
  • slink.pl (シンボリックリンク作成ツール)
  • sperl.pl (Perl テンプレート作成ツール)

utperl (ランチャ)

shtml.pl, slink.pl, sperl.pl を起動するためのランチャ。bash で書かれている。


slink.pl

Pl365Lib のモジュールをコピーしたり、シンボリックリンクを作成するためのツール。


sperl.pl

Pl365Lib を使った Perl プログラム(スケルトン)を作成するためのツール



sthml.pl (HTML 作成ツール)

WebPage で作る CGI の HTML テンプレートファイル(スケルトン)を作成するためのツール。
